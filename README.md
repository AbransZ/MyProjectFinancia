# Financia - Gestión de Finanzas Personales 📱

### Descripción
Financia es una aplicación nativa para Android diseñada para el control eficiente de gastos e ingresos. El objetivo principal es ofrecer una herramienta organizada y metódica para la salud financiera del usuario.

## 🛠 Stack Tecnológico
Para este proyecto se utilizaron las herramientas más modernas del ecosistema Android:

 * **Lenguaje:** Kotlin.
 * **Interfaz (UI):** Jetpack Compose (Desarrollo declarativo).
 * **Inyección de Dependencias:** Dagger Hilt.
 * **Red / API:** Retrofit con Corrutinas de Kotlin para llamadas asíncronas.
 * **Base de Datos:** Firebase (Firestore,Authentication,etc).

---

## 🏗 Arquitectura y Patrones
La aplicación está construida siguiendo principios de ingeniería de software para asegurar que sea escalable:

1. **Patrón de diseño:** MVVM (Model-View-ViewModel) para separar la lógica de negocio de la interfaz.
2. **Reactividad:** Uso de flujos de datos para actualizaciones en tiempo real.
3. **Clean Code:** Código organizado por capas (Data, Domain, UI).

---

## 🚀 Cómo ejecutar el proyecto
1. Clonar el repositorio: `git clone https://github.com/AbransZ/MyProjectFinancia.git`.
2. Abrir en **Android Studio** (Versión Ladybug o superior recomendada).
3. Sincronizar Gradle y ejecutar en un emulador o dispositivo físico.
